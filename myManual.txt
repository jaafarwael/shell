- These shell commands are defined internally:
- 1_cd <path> :change the current working directory
- 2_dir <directory> : list the cuntent of the directory if No directory is 
  specified list the current directory.
- 3_quit<> : exit from the Shell.
- 4_clr<> : clear the screen.
- 5_environ: list all enviroments Variable.
- 6_pause : stop the shell immediatly.
- 7_myShell <file>:execuates all the command withen the File.
- 8_./myShell <file>:execuates all the command withen the File.
- 9_command args > file: redirect output to file.                                                       
- 10_command args < file : redirect input to command from file.
- 11_command args < file1 > file2: redirect input from file1 to command and 
  redirect output from command to file2.
- 12_command args &: execuates command in the BackGround.
- 13_help<> to desplay this file.

- display help about any Command use : command --help.
